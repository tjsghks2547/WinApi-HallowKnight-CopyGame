# 📖기술 스택 및 개발 환경
- WinApi
- C++
  

# 💻구현 컨텐츠 및 기능 설명 

1. 플레이어
   - 키보드 입력 기반 이동, 회피 등 기본 조작 및 스킬 사용(공격/스킬 패턴, 쿨타임 관리 등) 구현

2. UI
   - HP / MP / 스킬 등 전투 정보를 표시하는 인게임 UI 및 상태 표시 요소 구현

3. 보스 및 몬스터
   - 여러 패턴을 가진 보스 AI 설계(패턴 전환, 패턴별 공격 방식, 피격/사망 처리 포함)

4. 스프라이트 애니메이션
   - 스프라이트 시트 기반 캐릭터/몬스터 애니메이션 재생(프레임 단위 애니메이션, 방향 전환 등) 구현

5. 이펙트
   - 스킬 사용, 피격, 적 처치 등에 연동된 시각 효과(이펙트 스프라이트, 간단한 연출) 구현
